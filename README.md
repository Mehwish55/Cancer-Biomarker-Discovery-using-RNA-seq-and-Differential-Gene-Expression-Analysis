# Cancer Biomarker Discovery using RNA-seq and DESeq2

## Overview

This project performs end-to-end differential gene expression analysis to identify potential cancer biomarkers using RNA sequencing (RNA-seq) data.

The workflow includes RNA-seq data preprocessing, differential expression analysis, visualization, functional enrichment analysis, and biomarker identification.

---

## Objectives

- Import RNA-seq count data
- Perform differential gene expression analysis using DESeq2
- Visualize expression patterns
- Identify significantly upregulated and downregulated genes
- Perform Gene Ontology (GO) enrichment analysis
- Perform KEGG pathway enrichment analysis
- Identify potential cancer biomarkers
- Generate publication-quality figures
